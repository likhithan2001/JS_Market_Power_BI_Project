# 📊 JS Market Analysis Dashboard (Power BI)

## 🚀 Project Overview
This project focuses on analyzing market sales data using **Power BI** to uncover actionable business insights. The dashboard provides a comprehensive view of sales performance, profitability, and regional trends to support data-driven decision-making.

---

## 🎯 Objectives
- Analyze sales and profit trends
- Identify top-performing products and regions
- Track KPIs for business performance
- Enable interactive data exploration

---

## 🛠️ Tools & Technologies
- Power BI
- Power Query (Data Cleaning & Transformation)
- DAX (Data Analysis Expressions)
- Excel / CSV Dataset

---

## 📊 Key Features
- Interactive dashboard with filters and drill-down capabilities
- KPI tracking (Total Sales, Profit, Growth)
- Region-wise and product-wise analysis
- Dynamic visualizations for better insights

---

## 📈 Key Insights
- Identified top-performing regions contributing to overall revenue
- Highlighted high-profit and low-profit product categories
- Discovered trends in sales growth over time
- Provided insights for improving business performance

---

## 🔍 Data Processing Steps
1. Data Collection from raw dataset  
2. Data Cleaning using Power Query  
3. Data Transformation and data modeling  
4. Creating calculated measures using DAX  
5. Building interactive dashboards
